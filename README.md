# Phishing Detection System

## 📌 Project Description
This project is a Real-Time Multi-Channel Phishing Detection System using Machine Learning.

It detects phishing content from:
- URLs
- SMS Messages
- Emails

## 🚀 Features
- Real-time prediction
- Risk score generation
- Data analysis charts
- Model performance metrics

## 🛠 Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas & NumPy
- HTML, CSS, JavaScript

## ⚙️ How to Run
1. Install dependencies
2. Run app.py
3. Open http://127.0.0.1:5000/

## 📊 Modules
- URL Detection
- SMS Detection
- Email Detection
