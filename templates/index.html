Project Information

Project Overview

Our project, the Phishing Detection System, is a machine learning based web application designed to detect phishing attacks in URLs, SMS messages, and Emails.

The system analyzes input data using trained machine learning models to determine whether the content is legitimate or malicious.

Project Features

  • URL Phishing Detection
  • SMS Spam / Smishing Detection
  • Email Phishing Detection
  • Risk Score Analysis
  • Dataset Visualization
  • Model Performance Analysis
